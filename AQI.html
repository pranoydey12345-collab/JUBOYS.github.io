About the Project

This project develops a low-cost real-time AQI monitoring system using ESP32, PMS5003, MQ135, and DHT11. The system measures PM2.5, PM10, harmful gases, temperature, and humidity, then uploads the live sensor data to Google Firebase using Wi-Fi. A connected mobile app reads this database and displays the air quality parameters in real time.

The aim is to raise awareness of pollution levels and support health precautions through continuous monitoring. This system is useful for environmental studies, urban monitoring, and public health applications.
